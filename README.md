# Request Cuti

Laravel-based Leave Management System.

## Features

- Employee Login & Register
- Leave Application
- Leave Types
- Leave Approval
- Dashboard
- Company Management
- User Management

## Installation

```bash
composer install
cp .env.example .env
php artisan key:generate
php artisan migrate --seed
php artisan serve
```

## Developed using

- Laravel
- PHP
- MySQL
- Bootstrap
